how to use

Visit any url with 2 numbers in it (http://localhost:8000/30/20/ for instance).

The html page generated there will show a list of numbers, starting from the first number (30 in this case), and showing incremental numbers from this point, the amound of numbers limited by the second number in the url (20). The list items for each number will also have a data-hook-id corresponding the same number
